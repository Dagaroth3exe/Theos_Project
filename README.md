## 🚀 Getting Started

### Clone the repo

```bash
git clone https://github.com/yourusername/theos-patisserie.git
cd theos-patisserie
```

### Install dependencies

```bash
npm install
```

### Run the project

```bash
npm start
```

Then open [http://localhost:4200](http://localhost:4200) in your browser.
